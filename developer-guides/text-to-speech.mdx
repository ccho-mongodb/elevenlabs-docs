## Overview

The ElevenLabs Text to Speech audio API provides a speech endpoint based on our TTS (text-to-speech) model. It comes with 6 built-in voices and can be used to:

- Create immersive audiobooks in 32 languages
- Generate natural-sounding voice-overs
- Deliver real-time conversational AI responses
