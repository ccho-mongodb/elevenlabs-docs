The ElevenLabs API provides a simple interface to state-of-the-art AI [models](/developer-guides/models) and [features](/api-reference/overview). Follow this guide to learn how to generate life-like speech, change voices, generate sound effects, isolate voices, generate voices & dub videos.

## Create and export an API key

Create an API key in the dashboard here, which you'll use to securely access the API.

With any plan, including the free plan, you gain full access to the ElevenLabs API. This allows you to interact with our [entire range of features](/api-reference/overview).

<Info>

    Both the API and the website draw from a shared quota pool. Usage on either platform will impact your available quota.

</Info>

## Authentication

To use the ElevenLabs API, include an xi-api-key header with your API key in all requests. Client SDKs set this header automatically, but if you're making manual requests, add it yourself.

<Steps>
    <Step title="Create an Account">
        To get your API key, [create an account](https://elevenlabs.io/sign-up), log
        in, and click on `API Keys` in the bottom left corner of the console.
    </Step>
    <Step title="Create an API Key">
        <Frame>![API Key](/api-reference/images/api_key.webp)</Frame>
        <Warning>

            Your API key is a secret. Do not share it or expose it in client-side code. Load it securely from an environment variable or key management service on your backend server.

        </Warning>
    </Step>

</Steps>
